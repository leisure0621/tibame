- Exception 抓取錯誤 String 是為了錯誤發生時，將錯誤信息存到物件中，統一處裡錯誤訊息，使處裡錯誤的人員能夠呼叫 getMessage()，並做後續處理
  - 取出當時發生 Exception 時的內容
- 使用 RuntimeException 處裡錯誤
  - 以不使用 try…catch 進行處理，但是如果有異常產生，則異常將由 JVM 進行處理。

# 命名規則

- 由大到小
- 小寫為套件名稱
- emp 套件名稱
- com.emp.
