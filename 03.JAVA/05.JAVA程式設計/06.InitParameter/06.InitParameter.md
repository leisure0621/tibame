   1. 因為換了一個新的 class loader(類別載入器)
   2. **因為換了新的行政院長**
2. 有 destroy 就要呼叫
   1. 沒有 destroy 就 do nothing
3. 如果有 init() 就要用 destroy 將資源還原回去
   1. 類似背景執行續，有開就要把它給關閉
4. 如果按 Terminate 就不執行 destroy
