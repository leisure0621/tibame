# 題目

- CSS3-Animations

# 作法

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <title>CSS3-Animations</title>
    <style type="text/css">
      body {
        background-color: #eba6a8;
        font-family: Arial;
      }

      #s1 {
        width: 500px;
        height: 50px;
        background-color: #336;
        color: #ffff66;
        font-size: 26px;
        margin: 10px;
        text-align: center;
        line-height: 50px;
        border-radius: 5px;
        animation: moveRight 3s infinite linear alternate;
      }
      @keyframes moveRight {
        form {
          transform: translate(0, 0);
        }
        to {
          transform: translate(500px, 0);
        }
      }
      #s2 {
        width: 500px;
        height: 50px;
        background-color: purple;
        color: orange;
        font-size: 26px;
        margin: 10px;
        text-align: center;
        line-height: 50px;
        border-radius: 5px;
        margin-left: 65%;
        animation: moveLeft 3s infinite linear alternate;
      }
      @keyframes moveLeft {
        form {
          transform: translate(0, 0);
        }
        to {
          transform: translate(-300%, 0);
        }
      }
      #s3 {
        text-align: center;
        font-family: Arial;
        font-size: 50px;
        letter-spacing: 1px;
        color: red;
        text-shadow: 2px 2px 2px brown;
        margin: 50px auto;
        animation: jumpShow 0.5s infinite linear alternate;
      }
      @keyframes jumpShow {
        0% {
          transform: scale(0, 0);
        }
        20% {
          transform: scale(1, 1);
        }
        30% {
          transform: scale(1, 1);
        }
        50% {
          transform: scale(0.5, 0.5);
        }
        55% {
          transform: scale(0.5, 0.5);
        }
        75% {
          transform: scale(0.3, 0.3);
        }
        80% {
          transform: scale(0.3, 0.3);
        }
        90% {
          transform: scale(1, 1);
        }
        95% {
          transform: scale(1, 1);
        }
        100% {
          transform: scale(0.5, 0.5);
        }
      }
    </style>
  </head>

  <body>
    <div id="s1">==CSS ANIMATION==</div>
    <div id="s2">==CSS ANIMATION==</div>
    <div id="s3">==SHOW TIME==</div>
  </body>
</html>
```
