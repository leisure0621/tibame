網頁設計

HTML

HTML是網頁的組成架構!沒有HTML也就無法完成一個網頁!而HTML是一種標籤定義的語言，也就是由國際標準組織W3C制定規範來讓設計網頁有一個參考依循。簡單說HTML就像是注音符號一樣，熟悉注音符號的發音與使用時機，我們就可以輕鬆對話。

CSS

CSS是美化網頁元素的魔術師，現在網頁都需要CSS來進行頁面的排版與美化。同樣的，CSS也是被定義規範的標籤，只要熟悉語法與屬性的使用，就可以打造出美麗的頁面。CSS 的全名為 Cascading Style Sheets，是一種樣式表 (Stylesheet) 語言。它的目的是為了對像 XHTML 及 HTML 之類的標記語言 (markup language) 提供一個顯示效果。有了 CSS，我們就可以將資料層及顯示層分開：HTML 標籤負責頁面內容的架構，而 CSS 則是負責頁面的顯示外觀。

jQuery

jQuery是javascript的函式庫，可以讓我們在設計頁面互動變化時，節省許多時間，許多頁面上的特效在之前都要使用很多敘述行來完成，現在有了jQuery這個函式庫，就可以節省開發時間!
