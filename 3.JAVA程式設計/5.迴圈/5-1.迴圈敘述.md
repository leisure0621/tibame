# <a id="s1" class="md-title" href="#top">1. 迴圈敘述</a>

- Java 迴圈方法

  - for( `初值設定` ; `條件判斷` ; `計次` ){ `敘述` };

    - 單行跑完就計次

  - while( `條件判斷` ){ `敘述` };

    - 跑到最下方才計次

  - do{ `敘述` }while( `條件判斷` );

    - 跑到下方後才計次，但第一圈會先跑

- 迴圈三要素

  - 初始值設定

    - 設定迴圈起始值

  - 條件判斷

    - 判斷是否重複執行迴圈的依據

      - 直到條件為 false 才跳出迴圈

  - 計次

    - 經過一次迴圈就計算一次，常用遞增或遞減
