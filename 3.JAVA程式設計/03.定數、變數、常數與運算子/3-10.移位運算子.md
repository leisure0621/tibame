# <a id="s1" class="md-title" href="#top">1. 移位運算子</a>

- `<<` : 左移运算符，num << 1,相当于 num 乘以 2

- `>>` : 右移运算符，num >> 1,相当于 num 除以 2

- `>>>` : 无符号右移，忽略符号位，空位都以 0 补齐
