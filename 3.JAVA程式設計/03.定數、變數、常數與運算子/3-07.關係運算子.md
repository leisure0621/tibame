# <a id="s1" class="md-title" href="#top">1. 關係運算子</a>

- `<` = `小於`

- `<=` = `小於等於`

- `==` = `等於等於`

- `!=` = `不等於`

- `>=` = `大於等於`

- `>` = `大於`

# <a id="s2" class="md-title" href="#top">2. 實際用例</a>

- ```java
  public class TestRelationalOP {
    public static void main(String[] args) {
      int num1 = 5, num2 = 3;
      System.out.println(num1 < num2);
      System.out.println(num1 <= num2);
      System.out.println(num1 == num2);
      System.out.println(num1 != num2);
      System.out.println(num1 >= num2);
      System.out.println(num1 > num2);
    }
  }
  ```
