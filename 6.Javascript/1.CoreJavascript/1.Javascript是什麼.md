# Javascript?

- 是一種直譯式語言

- 是一種單執行敘 (而 java 是一種多執行敘)

# 機制

- 自動回收記憶體

# 執行三步驟

- 語法分析
- 預編譯
- 解釋執行

  - 編譯一行執行一行

# 放置位置

- head
- body

# 放置方法

- `<script></script>`

- `<script src=""></script>`

# 為什麼要用變數

- `變數`相當於`指向資料的標籤`

# 所有變數都是弱型別

- `var`

- `let`

- `const`

- `""` 字串型別

- `''` 基本型別

- `true/false` boolean
