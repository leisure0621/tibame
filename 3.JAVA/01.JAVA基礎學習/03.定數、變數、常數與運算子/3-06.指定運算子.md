# <a id="s1" class="md-title" href="#top">1. 指定運算子</a>

- `=` = `a=2` = `a=2`

- `+=` = `a+=2` = `a=a+2`

- `-=` = `a-=2` = `a=a-2`

- `*=` = `a*=2` = `a=a*2`

- `/=` = `a/=2` = `a=a/2`

- `% =` = `a%=2` = `a=a%2`

# <a id="s2" class="md-title" href="#top">2. 實際用例</a>

- ```java
  public class TestAssignOP {
    public static void main(String[] args) {
      int num = 1;
      num += 2;
      String s = "1";
      s += 2;
      System.out.println(num); // 3
      System.out.println(s); // 12
    }
  }
  ```
