2019 Volvo V40 T3安全智尚版

第八代Volkswagen Golf
第八代Volkswagen Golf依然搭載了VAG集團的MQB模組化底盤，外型雖然與第七代車型類似，但新世代Golf轉往較為剛硬的形象發展，而厚實的C柱、橫幅式水箱護罩仍持續運用於第八代Golf上，不過，全新的IQ.Light頭燈組則改為較具銳利感的設計，就連LED日行燈也以眉形樣式呈現。至於車尾的部分同樣換上新造型的尾燈，值得注意的是，原本置於左後尾燈下方的GOLF銘牌，藉由第八代車型改放於品牌新VW廠徽之下。 相較於外觀給人「好像也沒變太多」的感覺，第八代Volkswagen Golf在車室的改變卻相當明顯，導入了10.25吋與8.25吋的全數位儀表及中控台觸控式螢幕，就連大燈的控制鈕也整合至延伸到駕駛左前方的數位儀表面板上，且冷暖空調控制鍵也整合於中控台螢幕下方，達到減少實體按鈕的效果，而第八代Golf亦捨棄了傳統排檔桿，選擇以較巧小線控排檔紐設置於中船上，再搭配中控台上的木質飾板，營造出既簡約又相當有質感的車室氛圍。
2019 BMW 3-Series Sedan 330i M Sport
全新BMW G20 3系列運用融合時尚設計語彙的線條，為新世代運動房車帶來具體樣貌。整體尺碼較上代增長76mm、拓寬16mm，車身高度僅僅增加6mm，再搭配2851mm長軸距，整體姿態更為寬闊穩健，加上增加41mm的軸距同時使後座膝部空間提升許多。 延續3系列特有的長引擎蓋、長軸距、短前懸設計手法，加上全新雙腎型水箱護罩與四摺線引擎蓋，營造出修長駿逸的視覺意象。此外，新3系列擁有0.23 Cd超低風阻系數，無論是高速平穩、行駛風切、降低油耗等皆能發揮具體功效。除了卓越機能，美學設計更是不容錯過，3系列搭配尾燈採用L型3D立體曲面設計，搭配雙邊單出排氣尾管，如配上Luxury風格套件前保險桿左右兩側更會鑲上T字造型側下氣壩，展現跑格風範。
