# <a id="s1" class="md-title" href="#top">1. 建立表格</a>

- `Tables` 右鍵 > `New Table`

  <div style="text-align:center">
    <img src="./image/0-2_37.png">
  </div>

- 添加欄位等細項

  - NUMBER

    - 自訂精準數 NUMBER(總位數,小數位數)

  - CHAR

  - VARCHAR2

    - 不確定字元長度就選這個

  - DATE

    - 儲存日期

  - TIMESTAMP

    - 自動創建當日時間: `Default` > `CURRENT_TIMESTAMP` <br><br>

  <div style="text-align:center">
    <img src="./image/0-2_38.png">
  </div>
