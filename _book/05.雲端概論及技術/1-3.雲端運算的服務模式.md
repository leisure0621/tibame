# <a id="s1" class="md-title" href="#top">1. 雲端運算的服務模式(記住)</a>

- Infrastructure-as-a-Service(Iaas) 計算能力

  - 資料中心(Data Center)

- Platform-as-a-service(PaaS) 編程環境

  - 開發、運行、管理、監控

  - 少用，一旦用了就會被此平台綁住了

- Software-as-a-Service(SaaS) 雲端軟體 應用系統

  - 提供軟體服務，僅需打開瀏覽介面查看
