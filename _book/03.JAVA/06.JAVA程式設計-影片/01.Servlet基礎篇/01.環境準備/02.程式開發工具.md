# 下載 Eclipse 並安裝

- **下載並安裝**
  - 至官網下載 [Eclipse IDE for Enterprise Java Developers](https://www.eclipse.org/downloads/packages/)
  - 須注意下載下來的檔名需含 **jee** 才是我們所需的企業版
  - 解壓縮後即可啟用
- **設置 WorkSpace 目錄**
  - 在任意磁碟 ( <u>C:/</u> ) 中創建 <u>eclipse_workspace</u> 目錄
