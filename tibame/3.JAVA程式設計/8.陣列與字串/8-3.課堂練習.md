- 方法 1: 逐行撰寫操作

  ```java
  package xxx;

  public class PricticeArray {
    public void printArray(int[][] arr) {
      for (int i = 0; i < arr.length; i++) {
        for (int j = 0; j < arr[i].length; j++) {
          System.out.print(arr[i][j] + "\t");
        }
        System.out.println();
      }
      System.out.println();
    }

    public static void main(String[] args) {
      int x[][] = new int[3][3],
          y[][] = new int[3][3],
          z[][] = new int[3][3];

      // Step1: 入值
      for (int i = 0; i < x.length; i++) {
        for (int j = 0; j < x[i].length; j++) {
          x[i][j] = (int) (Math.random() * 31);
        }
      }

      for (int i = 0; i < y.length; i++) {
        for (int j = 0; j < y[i].length; j++) {
          y[i][j] = (int) (Math.random() * 31);
        }
      }

      for (int i = 0; i < z.length; i++) {
        for (int j = 0; j < z[i].length; j++) {
          z[i][j] = x[i][j] + y[i][j];
        }
      }

      // Step2: 顯示
      for (int i = 0; i < x.length; i++) {
        for (int j = 0; j < x[i].length; j++) {
          System.out.print(x[i][j] + "\t");
        }
        System.out.println();
      }
      System.out.println();

      for (int i = 0; i < y.length; i++) {
        for (int j = 0; j < y[i].length; j++) {
          System.out.print(y[i][j] + "\t");
        }
        System.out.println();
      }
      System.out.println();

      for (int i = 0; i < z.length; i++) {
        for (int j = 0; j < z[i].length; j++) {
          System.out.print(z[i][j] + "\t");
        }
        System.out.println();
      }
      System.out.println();
    }
  }
  ```

- 方法 2: 切割成方法以符合 OOP 概念 (將重複使用的功能切割成小的方法模塊)

  - refactor (重構): 方便理解、維護、擴充

  ```java
  package xxx;

  public class PricticeArray {

    // 設定陣列值
    public int[][] getRandomArray() {
      int arr[][] = new int[3][3];

      for (int i = 0; i < arr.length; i++) {
        for (int j = 0; j < arr[i].length; j++) {
          arr[i][j] = (int) (Math.random() * 31);
        }
      }
      return arr;
    }

    // 列印陣列元素
    public void printArray(int[][] arr) {
      for (int i = 0; i < arr.length; i++) {
        for (int j = 0; j < arr[i].length; j++) {
          System.out.print(arr[i][j] + "\t");
        }
        System.out.println();
      }
      System.out.println();
    }

    public static void main(String[] args) {
      PricticeArray sa = new PricticeArray();
      int x[][] = sa.getRandomArray(),
          y[][] = sa.getRandomArray(),
          z[][] = new int[3][3];

      for (int i = 0; i < z.length; i++) {
        for (int j = 0; j < z[i].length; j++) {
          z[i][j] = x[i][j] + y[i][j];
        }
      }

      sa.printArray(x);
      sa.printArray(y);
      sa.printArray(z);
    }
  }
  ```
